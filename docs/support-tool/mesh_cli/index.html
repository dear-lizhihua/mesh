@mesh/cli

整体目标#

一个命令行工具，辅助开发

参考#

https://cloud.google.com/sdk/gcloud/reference

mesh env#

创建 .gitignore 文件#

$ mesh env create gitignore

默认配置

# editor
.idea
# misc
.DS_Store
# node
node_modules
# build
dist
build
# npm
npm-lock.json
npm-debug.log*
# yarn
yarn.lock
yarn-debug.log*
yarn-error.log*

.gitignore 文件添加目录或文件#

$ mesh env add gitignore \
.env.local \
.env.development.local \
.env.test.local \
.env.production.local

在 .gitignore 文件中添加如下四行

...
# ignore directory or file
.env.local
.env.development.local
.env.test.local
.env.production.local

创建 jsconfig.json 文件#

$ mesh env create jsconfig
$ mesh env create jsconfig --vite # 如果是 Vite 项目

默认配置

{
"compilerOptions": {
"baseUrl": ".",
"paths": {
"@/*": [
"src/*"
]
}
},
"exclude": [
"dist",
"node_modules"
],
"include": [
"src/**/*"
]
}

创建 LICENSE 文件#

$ mesh env create license # 默认 MIT
$ mesh env create license --cc4 # CC 4.0
  • MIT License
  • Creative Commons Attribution 4.0 International (CC BY 4.0)

创建 .prettierrc 文件和 .prettierignore 文件#

$ mesh env create prettierrc

.prettierrc 文件

{
"singleQuote": true,
"semi": false,
"printWidth": 120,
"trailingComma": "none",
"bracketSpacing": true,
"jsxBracketSameLine": false,
"arrowParens": "avoid",
"tabWidth": 2,
"useTabs": false
}

.prettierignore 文件

.idea
.dist
node_modules
**/node_modules

创建 .editorconfig 文件#

$ mesh env create editorconfig

默认配置

# EditorConfig is awesome: http://EditorConfig.org
# top-most EditorConfig file
root = true
# Unix-style newlines with a newline ending every file
[*]
end_of_line = lf
insert_final_newline = true
# Matches multiple files with brace expansion notation
# Set default charset
[*.{js,py}]
charset = utf-8
# 4 space indentation
[*.py]
indent_style = space
indent_size = 4
# Tab indentation (no size specified)
[*.js]
indent_style = space
# Indentation override for all JS under lib directory
[lib/**.js]
indent_style = space
indent_size = 2
# Matches the exact files either package.json or .travis.yml
[{package.json,.travis.yml}]
indent_style = space
indent_size = 2

mesh env init 初始化项目#

$ mesh env init

等同于

$ mesh env create gitignore
$ mesh env create jsconfig
$ mesh env create prettierrc
$ mesh env create editorconfig

mesh api#

接口转换为调用代码

$ mesh api generate --src=<source directory> --dist=<dist directory>
